## 如果下载链接打不开

下载链接依赖 GitHub Actions 自动生成安装包。首次上传项目后，请先执行：

1. 打开仓库顶部的 **Actions**。
2. 选择 **Build desktop installers**。
3. 点击 **Run workflow**。
4. 等待构建完成。
5. 回到本页面点击对应系统的下载链接。

也可以打开仓库右侧 **Releases** 区域，进入最新版本下载安装包。
